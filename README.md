## ✨ What is this?

**AI Software Company** is a web app where you manage a team of 5 AI-powered employees — each with their own role, personality, and expertise. Instead of chatting with one generic AI, you delegate tasks to specialists just like a real company.

Think of it as your **personal AI dev team available 24/7.**

## ⚡ Run Locally in 3 Steps

### 1. Clone the repo
```bash
git clone https://github.com/Ehtasham-Yasin/ai-company.git
cd ai-company
```

### 2. Get a free Groq API key
- Go to [console.groq.com](https://console.groq.com)
- Sign up free (no credit card needed)
- Create an API key and copy it

### 3. Add your key and open
Open `index.html` in VS Code, find this line:
```js
const GROQ_API_KEY = 'ENTER_YOUR_GROQ_KEY_HERE';
```
Paste your key, save, then open with Live Server or:
```bash
npx serve .
```
Visit `http://localhost:3000` — your AI team is ready! 🎉
